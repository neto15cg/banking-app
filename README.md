# BankingApp

BankingApp it's a layout created in React Native using [EXPO](https://expo.io) with objective of study.

# Installation

For install all packages use this command:

```bash
npm install
```

or

```bash
yarn install
```

## Usage

```bash
expo start
```

or

```bash
yarn start
```
